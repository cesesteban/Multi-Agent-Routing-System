# Política de Hardware Corporativo

## Asignación de Equipo
- Laptops: Se asignan de acuerdo al rol de cada empleado (Estándar, Alta Capacidad).
- Dispositivos de Salud: Se provee teclado y mouse ergonómicos a solicitud.

## Mantenimiento y Reposición
- Período de renovación: 3 años.
- En caso de daño accidental, notificar a Tecnología para reparación o reemplazo.
- Pérdida o Robo: Reportar de inmediato a soporte y realizar la denuncia correspondiente si aplica.

## Uso Personal del Hardware
El equipo es para uso profesional exclusivo. No se permite la instalación de software no autorizado o juegos personales.
