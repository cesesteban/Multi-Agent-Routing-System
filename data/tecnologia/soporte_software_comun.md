# Soporte de Software Corporativo y Aplicaciones Comunes

## Suite de Oficina (Microsoft 365)
- Outlook: Para correo y calendario.
- Teams: Para video llamadas y chat.
- Excel/Word: Para edición de documentos.

## Repositorio de Código (GitHub)
- Acceso a repositorios mediante llaves SSH.
- Flujo de trabajo basado en Pull Requests (Standard PR Flow).
- Revisión de código obligatoria por al menos 1 par.

## Gestión de Proyectos (Jira)
- Todos los tickets de soporte deben crearse en el tablero de IT.
- Las incidencias de alta prioridad (P1) se notifican automáticamente por Slack.

## Software No Autorizado
- No se permiten navegadores fuera de Chrome/Edge si no es para pruebas.
- Prohibida la instalación de software de mensajería personal en computadoras corporativas.
